# OpenFSD_SunnyPilot

A sunnypilot branch to use other pandas via a generic CAN Car with experimental FlexRay support.

## Features

- **FlexRay Protocol Support**: Experimental implementation of FlexRay communication protocol
- **Generic CAN Car**: Support for various Panda devices via generic CAN interface
- **Simulator**: Software simulator for testing without hardware
- **Examples**: Sample code and documentation

## Quick Start

Run the FlexRay example:

```bash
python examples/basic_flexray.py
```

## Documentation

See [docs/FLEXRAY.md](docs/FLEXRAY.md) for detailed FlexRay documentation.
